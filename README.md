### SDL3 | 2D Pixel experiment

This is an SDL3 experiment in C.

## Requirements & dependencies
1. Make sure you first download the SDL3 dependency. It should be placed in the vendored/SDL folder
```git clone https://github.com/libsdl-org/SDL.git vendored/SDL ```
2. You need cmake (idk version) to compile

## Installation
If all is well then you should just have to run cmake and then build&run the program.

```cmake . && make && ./build/sdl3test```
